# Local Guide Project Context

This is The Local Guide - Chennai Edition, a project that demonstrates local knowledge and cultural understanding through multiple interfaces.

## Project Structure
- `product.md` - Complete local knowledge base with Chennai culture, Tamil slang, food, and traffic
- `local_guide.py` - Interactive CLI tool that parses and uses the knowledge base
- `index.html` - Web interface for the local guide

## Key Features
- Local slang translation (Tamil/English mix)
- Food recommendations based on preferences
- Traffic pattern analysis with real-time considerations
- Weather and monsoon information
- Cultural etiquette and local insights

## Development Guidelines
- Keep the local knowledge base in `product.md` as the single source of truth
- Python tool should parse and utilize the markdown content dynamically
- Web interface should provide an intuitive way to access all features
- Maintain Chennai authenticity - respect Tamil culture and local expressions!
